## Description:
Cette exposition avait un grand écran avec plusieurs chaises et on pouvait y jouer à plusieurs personnes. Le but est de se rendre à la destination demandé sans se faire toucher incluant les coins. Il y a plusieurs niveaux à ce jeu.
 
![photo](media/projection_visuelle.jpg)
>Cette photo du jeu a été prise par Amélie Pelletier

## Comment le jeu fonctionne (Matériel nécessaire)
- Pour commencer, il y a un code QR qu'ont peut utiliser pour rejoindre la partie.

- Des téléphones cellulaires.

- Le jeu est organisée comme un parcours interactif et immersif sur un grand écran.

- Un écran pour afficher les contenus interactifs.

- Des haut-parleurs pour l'audio.

![photo](media/codeqr.jpg)
>Le code QR du jeu prise par Rosa-Lee Savoie
 
 ## Type d'installation: 
- L'installation est interactive.

- Installation interactive où les visiteurs peuvent cliquer et interagir.

- Installation audiovisuelle qui contient des images, vidéos, animations et sons.

 ## Expérience vécu: 
 J'ai bien aimée l'ambiance de la salle. J'ai aussi aimé les personnages qu'ont étaient donnés et les sound effects.

## Ce qui m'a plu:
Ce qui m'a plu étaient les installations interactives qui permettent de mieux comprendre comment jouer avec les bouttons sur mon téléphone.

Dans la salle, il y avait un grand écran avec un code QR et plusieurs bancs où ont pouvaient s'assoirent.
